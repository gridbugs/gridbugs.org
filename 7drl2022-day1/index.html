7 Day Roguelike 2022: Bootstrapping

March 05, 2022

I spent most of the day bootstrapping the project. Unlike the last few years, this year I’ve elected to start from scratch rather than from an existing project, though I am copying code aggressively from last years entry. The reason for doing this is to prevent needing to prevent the technical debt from accumulating over multiple game jams hacking on the same code base.

So far you can move the player character around, and open doors. Collisions work, as does visible area detection and dynamic, diminishing lighting. The game can be saved and loaded, it has a main menu and a pause menu, and controls can be changed by editing a config file.
