Tmux runs a login shell by default

November 07, 2020

Unless configured otherwise, shells running in tmux panes are login shells. This means that if you start tmux from a shell whose environment already reflects your (say) .profile script, you’ll end up sourcing it a second time leading to problems highlighted here.

To have tmux run a non-login shell, add this to .tmux.conf:

set -g default-command "${SHELL}"
